Basic Commands

Here you will learn the basic commands for setting up a Cisco router.

Router>

User configuration mode. Allows only a limited number of basic commands.

Router>logout

Exit Exec Mode

Router>enable

Allows you to enter Privileged Exec Mode

Router#

Privileged configuration mode. Allows access to all the Router's commands and functions.

Router#disable

Exit privileged mode

Router#configure terminal

Allows you to enter Global Configuration Mode

Router(config)#:

Global Configuration Mode. In which different sub-configuration modes of a specific part or function of the IOS device can be entered

Router(config-if)#:

Interface configuration mode. Used to configure a switch port or router network interface

Router(config-sub-if)#:

Sub Interface Configuration Mode

Router(config-line)#:

Line Configuration Mode (affects Line, Console, Vty, Auxiliary modes, etc.)

Router(config-router)#:

Router Configuration Mode

Router(config)#hostname RO

Allows you to change the name of the Device

RO(config)#no hostname

Permite eliminar el nombre de host configurado y regresar al nombre de entrada predeterminada
