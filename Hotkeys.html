Hotkey list

Here you'll learn keyboard shortcuts for setting up a Cisco router.

Tabulation

Complete a partial command name entry

Recoil

Delete the character to the left of the cursor

Crtl+A

Move the cursor to the beginning of the line

Crtl+D

Delete the character where the cursor is

Crtl+E

Move the cursor to the end of the command line

Ctrl+F or Right Arrow

Move the cursor one character to the right

Ctrl+K

Delete all characters from the cursor to the end of the command line

Ctrl+L or Ctrl+R

Redisplays the system input prompt and command line after a message is received from the console

Ctrl+P or Up Arrow

Re-enter the command found in the history buffer from the most recent commands

Ctrl+U or Crtl+X

Delete all characters from the cursor to the beginning of the command line

Ctrl+W

Delete the word to the left of the cursor

Esc D

Delete all characters from the cursor to the end of the word

Ctrl+C

When in setup mode, it interrupts and returns to the command prompt

Ctrl+Z

When in any configuration mode, it terminates the configuration mode and returns to privileged EXEC mode

Ctrl+Shift+6

It is used to interrupt DNS lookups, traceroutes, pings

Ctrl+B or Left Arrow

Move the cursor one character to the left
