Open Shortest Path First (OSPF)

Here you will learn the open shortest path first for configuring a Cisco router.

RO(config)#router ospf 10

Enable ospf routing protocol under ID 10 processes

RRO(config-router)#network 10.10.0.0 0.0.255.255 area 0

Enable ospf with Area 0 Matching Interface

RO(config)# interface loopback 0 10.10.0.0 0.0.255.255

Create a loopback interface

RO(config-if)#ip address 192.168.250.250 255.255.255.0

Assign the IP address to the loopback interface

RO(config-router)#router-id 1.1.1.1

Set 1.1.1.1 as the Router ID

RO(config-if)#ip ospf priority 200

Configuration to influence dr/BDR selection processes. The valid range is 0 to 255.0 makes the router intelligible for dr/bdr

RO(config-if)#bandwidth 256

Configuration to influence the metric cost of the route. Higher bandwidth has lower cost. Bandwidth is defined in kbps. 256 is 256 kbps

RO(config-if)#ip ospf hello-interval timer 20

Set hello time interval to 20 seconds. The greeting time must match on both routers to become neighbors

Router(config-if)#ip ospf dead-interval 50

Set the dead interval time to 50 seconds. The dead interval time must match on both routers to become neighbors

RO#show ip route ospfighbors detail

Displays all learned routers through the OSPF routing table

RO#show ip ospf

Muestra información básica sobre OSPF

RO#show ip ospf interface

Displays information about all active OSPF interfaces

RO#show ip ospf interface serial 1/1/1

Displays OSPF information about the 1/1/1 serial interface

RO#show ip ospf neighbor list all

Displays basic information of neighbors with OSPF

RO#show ip ospf neighbors detail

Displays detailed information of neighbors with OSPF

RO#show ip ospf database

Displays information from the OSPF database

RO#clear ip route *

Cleans up all routes in the routing table

RO#clear

Cleans up a particular route from the routing table

RO#clear ip opsf counters

Clean OSPF meters

RO#debug ip ospf events

Displays all OSPF events

RO#debug ip ospf packets

Displays changes to OSPF packages

RO#debug ip ospf adjacency

Displays DR/BDR state election processes
