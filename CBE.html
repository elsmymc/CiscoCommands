Basic Editing Commands

Here you will learn the basic editing commands for setting up a Cisco router.

RO#copy running-config startup-config

Allows you to save changes made to NVRAM

RO#wr

Short Command

RO#copy running-config tftp

Allows you to store active settings on a TFTP server

RO#copy tftp flash

Allows you to download a new image from a TFTP server to Flash memory

RO#copy flash tftp

Allows copying the system image from Flash memory to a TFTP server

RO#copy tftp running-config

Allows configuration information to be uploaded from a TFTP server

RO#reload

Allows you to reset startup settings and recharge your device

RO#erase startup-config

Clear the startup settings (NVRAM) and restart the device

RO#erase flash

Erase the contents of the Flash memory
