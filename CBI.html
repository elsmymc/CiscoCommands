Basic Interface Commands

Here you will learn the basic interface commands for configuring a Cisco router.

RO(config)#interface Gigabitethernet 0/0

Configure the interface

RO(config-if)#ip address 192.168.1.0 255.255.255.0

Assign IPv4 address

R0(config-if)#ipv6 address 2001:db8:acad:2::1/64

RO(config-if)#description LAN 1 binding

RO(config-if)#no shutdown

Enable the interface

RO(config-if)#ipv6 address fe80::1 link-local 2001:db8:acad:2::1/64

Assign a FE80::1 link-local address to an interface to easily recognize that it belongs to the R router

RO(config)# ip route 0.0.0.0 0.0.0.0 serial 0/0

Assign an IPv4 link-local address to an interface to easily recognize that it belongs to the R router

RO(config)# ipv6 route ::/0 s0/0/0

Assign IPv6 link-local address to an interface

RO(config)#ip default-gateway 172.16.0.1
