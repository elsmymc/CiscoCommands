Example of Network Address Translation (NAT)

Static NAT Configuration

This NAT configuration consists of mapping a private address to a public address statically.

Dynamic NAT Configuration

This configuration uses a Public IP Pool for a Private IP Pool that will be mapped dynamically and on demand.

NAT Configuration Overloaded or PAT

The PAT configuration sets the overhead translation for the NAT-POOL2 NAT set. NAT-POOL2 contains the addresses from 209.165.200.226 to 209.165.200.240. Hosts on the 192.168.0.0/16 network are subject to translation. The S2/0 interface is identified as the internal interface, and the S3/0 interface is identified as the external interface.
