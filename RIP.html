Routing Information Protocol (RIP)

Here you will learn the basic commands for configuring Routing Information Protocol (RIP) on your Cisco router.

RO(config)#router rip RO(config-router)#network B.C.D.E

Enable the RIP routing protocol. By default it enables version 1

RO(config-router)#version 2

Enable RIP Routing Protocol Version 2

RO(config-router)#no network B.C.D.E

Remove the IP address from the RIP routing protocol

RO(config)#no router rip

Disables the RIP routing protocol

RO(config-router)#no auto-summary

Disable automatic summarization

RO(config-router)#passive-interface s0/0/0

Enable RIP not to issue routing updates from this interface

RO(config-router)#ip Split-horizon

Enable split horizon

RO(config-router)#no ip Split-horizon

Disable split horizon (enable by default)

RO(config-router)#timers basic 30 90 180 270 360

Set the RIP timer in seconds. 30 (Routing Update), 90 (Invalid Timer), 180 (Hold Timer), 270 (Discharge Timer), 360 (Sleep Timer)

RO#debug ip rip

Allows you to see all RIP-related activity in real-time

SW(config-if)#switchport mode trunk

Changing the Interface Mode in Trunk

RO#show ip rip database

Displays the contents of the RIP private database

RO#show ip protocols

Displays the IPv4 routing protocol parameters currently configured on the router
