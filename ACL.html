Access Control List

Here you will learn the access control list for configuring a Cisco router.

Standard ACL

Only source address is specified

RO(config)#access-list 1 permit 10.0.0.0 0.255.255.255

Define who traffic is allowed

RO(config)#interface Fa0/1

Apply the ACL to the interface

RO(config-if)#ipaccess-group 1 out

Apply the ACL to the interface

RO(config)#access-list 102 deny tcp any any

Denies the rest of the traffic

Extended ACL

Protocol, source and destination address are specified

RO(config)#access-list 105 permit 10.5.4.0 0.0.0.255 host 10.5.64.30 eq80

Defines who and what protocol traffic is allowed to

RO(config)#interface serial 0/0

Apply the ACL to the interface

RO(config-if)#ipaccess-group 100 out

Apply the ACL to the interface

RO#show Access-lists

Display ACLs, including those used by NAT

RO#show Access-lists 100

Displays a specific ACL

RO#no Access-lists 100

Remove a specific ACL
