Basic Show Commands

Here you will learn the basic show commands for configuring a Cisco router.

RO#show arp

Displays the assignment of IP addresses to MAC to router interface

RO#show cdp

Displays the interval between CDP publications, validity time, and publication version

RO#show cdp neighbors

Displays information from neighboring Cisco devices running that protocol, exchanges information from directly connected device hardware and software

RO#show cdp neighbors detail

Displays the IP address of a neighboring device helps determine if one of the CDP neighbors has an IP configuration error

RO#show cdp traffic

Displays CDP counters, including the number of packets sent and received and checksum errors

RO#show clock

Displays the date and time of the device

RO#show controllers serial number

Displays important information such as what type of cable is connected

RO#show crypto key mypubkey rsa

Displays the public encryption key that was used for SSH

RO#show debugging

Displays information about the types of debugs that are enabled

RO#show dhcp lease

Displays information about the leased IP address

RO#show flash

Displays the layout and contents of the Flash memory

RO#show history

Displays the history of commands that are currently in the history buffer

RO#show hosts

Displays a cached list of hostnames and addresses

RO#show interface

Displays statistics for all interfaces on a device

RO#show interface serial 0/0

Displays detailed information of the specified interface, status, protocol, duplex, speed, encapsulation, traffic of the last 5 minutes

RO#show interfaces description

Displays the description of all interfaces

RO#show interfaces status

Displays the status of all interfaces such as speed, duplex, trunk or Access vlan, connection

RO#show ip interface

Displays IPv4 statistics for all interfaces on a router

RO#show ip interface brief

Displays an overview of all interfaces, IPV4 address, their physical status, protocol status, and IP address

RO#show ip protocols

Displays the parameters and current status of the active routing protocol process

RO#show mac-address-table

Muestra las entradas de la tabla de direcciones MAC

RO#show memory

Displays statistics about router memory

RO#show process

Displays information about active processes

RO#show protocols

Displays configured Layer 3 protocols

RO#show running-config

Displays the current configuration file stored in DRAM. Loses all content when the device is turned off or restarted

RO#show running-config interface

Displays the commands configured in the specified interface

RO#show sessions

Displays Telnet connections established on the router

RO#show startup-config

Displays the NVRAM-stored configuration file that is used in the first boot process

RO#show version

Displays information about the Router and its interfaces, RAM, NVRAM, flash, IOS, IOS version, etc.

SW#show port-security

Overview of Port Security of All Interfaces

SW#show port-security interface fa0/5

Displays detailed information about port security on the specified interface
