Network Address Translation (NAT)

Here you will learn the networ address transation for configuring a Cisco router.

RO(config)#access-list 3 permit 192.168.1.0 0.0.0.255

Create an ACL that determines that the IP address can be translated (Dynamic NAT)

RO(config)#ip nat poll PUB 200.1.1.1 200.1.1.6 netmask 255.255.255.248

Create a public IP address survey (Dynamic NAT)

RO(config)#ip nat inside source list 3 poll PUB

Set up a NAT statement

RO(config)#ip nat inside source list 3 pool PUB overload

Using Keyword Overload at the End of NAT Statement

RO#show ip nat stasitics

Display counters for packages and native table entries, as well as basic configuration information

RO(config-if)#interface serial 0/0/0/0.1 point-to-point

Create a Peer-to-Peer Subinterface Numbered 1

RO#show ip nat translations

Show the NAT table

RO#clear ip nat translations *

Clean up all dynamic entries in the NAT table

RO#debug ip nat

Issues a log message to describe each packet whose IP address is translated with NAT
