Enhanced Interior Gateway Routing Protocol (EIGRP)

Here you will learn the enhanced interior gateway protocol for configuring a Cisco router.

RO(config)#router eigrp 20

Enable EIGRP with the number AS 20. The AS number must be the same on all routers to become an EIGRP neighbor

RO(config-router)#network 10.10.0.0

Enable EIGRP on interfaces that belong to the 10.0.0.0/8 network. (Classy Implementation)

RO(config-router)#network 10.10.0.0 0.0.255.255

Enable EIGRP on the interface that belongs to the 10.10.0.0/16 network. (Classless Implementation - Wildcard Mask Method)

RO(config-router)#no network 10.10.0.0

Disable EIGRP on interfaces that belong to the 10.0.0.0/8 network

RO(config-router)#no auto-summary

Turn off EIGRP's auto-summarization feature

RO(config)#no router eigrp 20

Disable the EIGRP 20 Routing Protocol

RO(config)#bandwidth 64

Set bandwidth to 64 Kbps
